## Data Flow

1. **Initial Load:** `PageLoader.tsx` displays a calibrated 5-second animation while assets are prepared.
2. **Layout Hydration:** `RootLayout` initializes fonts, smooth scrolling (Lenis), and the global `Navbar`/`Footer`.
3. **Route Rendering:** The Next.js App Router serves page components from `src/app/`, fetching data from `src/data/` or `src/lib/`.
4. **Client Interactivity:** GSAP and Framer Motion handle scroll-triggered animations and mouse-driven parallax effects.
