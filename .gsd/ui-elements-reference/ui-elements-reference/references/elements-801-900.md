## Category 81: Health & Wellness UI (#801–#810)

**#801 Activity Ring / Progress Circle**
Concentric circular rings (Apple Watch-style) showing progress toward daily goals: Move, Exercise, Stand — each ring fills proportionally with a color change at 100%.
Impact: Goal progress readable at a glance — numeric percentage less motivating than visual fill.

**#802 Symptom Checker Flow**
A guided step-by-step symptom selection interface with body diagram tap-points, symptom severity sliders (1–10), duration inputs, and triage output (See a doctor / Home care / Emergency).
Impact: Medical triage self-served — reduces unnecessary urgent care visits and improves appropriate care-seeking.

**#803 Medication Tracker Card**
A daily medication schedule card showing: medication name, dosage, schedule time, and a check-off button. Missed doses highlighted in amber. Streak counter for adherence.
Impact: Medication adherence improved when daily schedule is visualized and checkable.

**#804 Biometric Trend Chart**
An annotated line chart for health metrics (weight, blood pressure, glucose, sleep) with reference range bands (normal zone in green, caution in yellow, alert in red) and trend annotation.
Impact: Health trajectory immediately readable — single data points meaningless without trend context.

**#805 Mood / Energy Log Picker**
A quick daily mood or energy log using an emoji scale or colored slider — one-tap entry minimizes friction for daily habit formation. Historical trends shown as a calendar heatmap.
Impact: Daily logging maintained when entry requires under 5 seconds.

**#806 Appointment Booking with Provider Profiles**
A scheduling interface showing: provider photo, name, specialization, next availability, patient rating, and accepted insurance — with calendar slot selection.
Impact: Healthcare booking as smooth as restaurant reservations — provider context enables confident selection.

**#807 Health Data Privacy Toggle**
Explicit per-category toggles for which health data categories are shared: activity, nutrition, sleep, vitals, mental health — each with a plain-language description of how data is used.
Impact: Health data consent granular and informed — opaque bulk consent replaced by category-level control.

**#808 Nutrition Barcode Scanner UI**
A camera-triggered barcode scanner that looks up scanned food products in a nutrition database and adds them to the daily food log with a portion size selector.
Impact: Manual nutrition logging replaced by scan — data accuracy improves with barcode lookup.

**#809 Water Intake Tracker**
A visual water glass or bottle that fills incrementally as the user logs cups/glasses. Animation on log entry. Daily goal progress displayed as percentage and glasses count.
Impact: Hydration habit formed through visual progress — abstract "drink more water" advice actionable.

**#810 Step Counter Widget**
A circular or linear step progress indicator showing steps taken vs. daily goal — with a distance equivalent and calorie estimate. Integrates with device motion sensors via Sensor API.
Impact: Physical activity awareness drives increased movement — passive monitoring without manual entry.

## Category 82: Education & Learning UI (#811–#820)

**#811 Course Progress Dashboard**
A dashboard showing: enrolled courses, completion percentage per course, current lesson, estimated time remaining, streak, and a "Continue Learning" primary CTA.
Impact: Learner momentum maintained — clear position and next step removes re-entry friction.

**#812 Video Lesson with Embedded Quiz**
A video player that pauses at configured timestamps to present an inline knowledge check quiz — resuming only after a correct answer. Combines passive watching with active recall.
Impact: Knowledge retention dramatically higher with active recall interruptions (Feynman effect).

**#813 Flashcard Deck UI**
A card-flip interaction for spaced-repetition study — showing a prompt on the front and revealing the answer on flip. Self-rating (Easy / Hard / Again) feeds the SRS algorithm.
Impact: Spaced repetition retention doubles compared to massed studying.

**#814 Code Playground (In-Lesson)**
An embedded interactive code editor within a lesson allowing students to write, run, and see output of code without leaving the learning platform.
Impact: Hands-on coding practice immediate — copy-to-local setup steps eliminated.

**#815 Assignment Submission UI**
A file upload or rich text editor for submitting assignments — with a deadline countdown, submission confirmation, and revision submission if reattempts are allowed.
Impact: Assignment workflow self-contained in the platform — no email attachment submission.

**#816 Peer Review Interface**
A structured review form where students review each other's work against a rubric — viewing submitted work alongside the scoring criteria with text feedback fields.
Impact: Learning through evaluation deepens understanding — and scales grading capacity.

**#817 Knowledge Map / Learning Path Visualizer**
An interactive node graph showing the skill tree or learning path — completed topics in green, current in highlight, locked future topics greyed out with unlock prerequisites listed.
Impact: Learning trajectory visualized — learners understand what's next and why order matters.

**#818 Annotation & Highlight on Reading Material**
Highlighting text in reading material with color categories, adding margin notes, and saving annotations to a personal notes view — synchronized across devices.
Impact: Active reading replaces passive — annotation engagement correlated with comprehension scores.

**#819 Live Class / Webinar UI**
A video conferencing layout with: speaker video prominent, raised-hand queue, live Q&A chat panel, polls widget, reactions (👍🎉❓), and participant count.
Impact: Virtual learning environment closely approximates in-person engagement dynamics.

**#820 Certificate & Achievement Display**
A shareable completion certificate with: learner name, course title, completion date, instructor signature, and a verification URL — downloadable as PDF, sharable to LinkedIn.
Impact: Completion celebrated and credentialed — external recognition drives course completion rates.

## Category 83: Finance & Banking UI (#821–#830)

**#821 Account Balance Dashboard**
A clean summary view showing all accounts (checking, savings, credit, investment) with current balances, last-updated time, and trend arrows — collapsible per account category.
Impact: Full financial picture in one view — multi-app checking replaced by unified dashboard.

**#822 Transaction List with Categorization**
A scrollable transaction list with: merchant logo, name, amount, date, and AI-assigned category tag (Food, Transport, Shopping) — filterable by category and date range.
Impact: Spending patterns visible without manual categorization — ML saves hours of bookkeeping.

**#823 Spending Breakdown Donut**
A donut chart showing spending allocation by category for the current month — with segments clickable to drill into transactions in that category.
Impact: Budget adherence visible at a glance — overspending categories identified immediately.

**#824 Budget Envelope Tracker**
Visual "envelopes" for budget categories showing: budget amount, spent amount, remaining amount, and a fill-progress bar — turning red as limit approaches.
Impact: Budget remaining always front-of-mind — spending-to-budget gap visible without mental math.

**#825 Bill Due Calendar**
A calendar view overlaying upcoming bill due dates and predicted account balance based on recurring income and expenses — flagging dates where balance may go negative.
Impact: Cash flow crises anticipated days in advance — overdrafts prevented by visibility.

**#826 Investment Portfolio Chart**
A time-series chart of portfolio total value with: period selector, benchmark comparison overlay, contribution markers, and allocation breakdown donut.
Impact: Investment performance contextualized — raw balance without benchmark is meaningless.

**#827 Recurring Transfer Scheduler**
A UI for setting up automatic transfers with: from/to account selector, amount, frequency (Weekly/Monthly/Custom), start date, end condition, and pause/cancel controls.
Impact: Savings automation configured visually — recurring savings goals set-and-forgotten.

**#828 Card Management UI**
A card display panel showing: card art image, last 4 digits, expiry, cardholder name — with controls to freeze/unfreeze, report lost, change PIN, view limits, and manage spend controls.
Impact: Card management self-served — no call center required for freezes or limit adjustments.

**#829 Money Transfer Flow**
A step-by-step transfer flow: select recipient (or add new) → enter amount with balance validation → review with fee display → confirm with biometric → success with receipt.
Impact: Transfer completion smooth and secure — five steps cover 100% of error cases.

**#830 Financial Goal Tracker**
A goal card showing: goal name (e.g., "Emergency Fund"), target amount, current amount, projected completion date, and a progress bar — with a "Add money" shortcut.
Impact: Abstract savings goals become tangible progress — visual progress motivates continued contribution.

**#833 Skills Proficiency Visualization**

Impact: Skill depth communicated visually — "Proficient in React" replaced by a meaningful indicator.

**#834 Timeline / Career Journey**

**#840 Dark / Light Portfolio Toggle**

Impact: Both portfolio and designer competency demonstrated simultaneously.

## Category 85: News & Media UI (#841–#850)

**#841 Breaking News Banner**
A high-contrast top banner with "BREAKING" badge, headline, and auto-refresh for developing stories — positioned above navigation with a close button for non-urgent removal.
Impact: Critical news reaches every visitor immediately — buried in the feed means missed.

**#842 Article Recommendation Engine**
An "If you liked this, read…" section after article completion — recommendations based on topic similarity, reading history, and trending signals — not just editorial picks.
Impact: Session depth increases when personalized recommendations match demonstrated interest.

**#843 Journalist / Author Profile Page**
A dedicated author page with: photo, bio, beat, article count, featured series, social links, and a follow button — building reader-journalist relationships over time.
Impact: Loyal readership attached to voices — not just to the outlet — drives retention.

**#844 Article Save / Bookmark for Later**
A bookmark icon on article cards and open articles saving to a personal reading list — accessible from the nav. Push notification reminders for unread saved articles.
Impact: Articles saved for later are actually read — read-it-later captures deferred reading intent.

**#845 Topic / Tag Follow**
Following a topic (Climate, AI, Finance) subscribes the user to a personalized feed of that topic's articles — notified when major stories break in followed topics.
Impact: Personalized topic streams replace homepage as primary entry point — engagement and loyalty increase.

**#846 Live Blog / Ticker**
A real-time updating blog for developing stories — newest entries appear at the top with timestamps, auto-refreshing on a short poll interval or WebSocket push.
Impact: Breaking story readers stay on-site — individual page refreshes replaced by auto-push updates.

**#847 Article Text Size / Font Preference**
A reading preference panel offering: font size (S/M/L/XL), typeface (serif/sans-serif), line width (compact/comfortable/wide), and high contrast toggle.
Impact: Reading experience personalized — accessibility and comfort served without a settings menu.

**#848 Paywall Meter (Metered Access)**
A visible counter showing free articles remaining ("3 of 5 free articles this month") — appearing after the third article with a soft paywall prompt while remaining 2 are accessible.
Impact: Soft paywall converts curious readers — hard paywalls repel everyone who hasn't already decided.

**#849 Comment Quality Sorting**
Comment sections sortable by: Top (upvotes), Newest, Staff picks, and Verified subscribers — defaults to quality-sorted rather than chronological to surface best discourse first.
Impact: Comment section quality visible — vitriolic chronological noise replaced by elevated discourse.

**#850 Offline Reading Mode**
A Service Worker-powered offline reading mode that caches articles opened while online for reading without connection — with an offline badge indicator when the user is disconnected.
Impact: Commuter and intermittent-connection readers served — offline gap in content consumption eliminated.

## Category 86: Travel & Booking UI (#851–#860)

**#851 Flexible Date Search**
A date picker showing a ±3 day flexibility toggle — "Flexible with my dates" mode displaying prices across a calendar grid to surface the cheapest travel windows.
Impact: Price-sensitive travelers book on optimal dates — rigid date search misses flexible travelers.

**#852 Destination Inspiration Grid**
A visual grid of destination cards with: photo, city name, price tag ("from $299"), and top-of-mind tags (Beach, City, Adventure) — for users without a fixed destination.
Impact: Undecided travelers guided to destinations matching interests — discovery replaces blank search.

**#853 Flight / Hotel Comparison Table**
Side-by-side comparison of 2–3 selected options with: price, duration, stops, amenities, ratings, cancellation policy — pinned for comparison from a search results list.
Impact: Selection confidence increases when options are directly compared — individual card browsing is slower.

**#854 Seat Map Selector**
An interactive aircraft or venue seating diagram showing seat availability (green), unavailable (grey), premium (gold) — with click-to-select and seat feature legend (extra legroom, window, aisle).
Impact: Seat selection informed by spatial layout — "choose any seat" anxiety replaced by visual selection.

**#855 Price Alert / Watchlist**
A "Track price" button on flights and hotels that triggers an email/push alert when the price drops below a threshold — keeping intent warm without booking prematurely.
Impact: Price-sensitive travelers book at right price — high prices no longer end the journey.

**#856 Trip Itinerary Builder**
A drag-and-drop day-by-day itinerary planner with: activity cards, time slots, map integration showing route between activities, and exportable PDF itinerary.
Impact: Trip planning self-contained — fragmented notes, screenshots, and tabs consolidated.

**#857 Check-In / Boarding Pass UI**
A mobile-first boarding pass display with: QR code, passenger name, flight number, gate, seat, boarding time — in a wallet-ready format with large text and high contrast.
Impact: Physical boarding pass replaced — airport friction eliminated for mobile check-in.

**#858 Hotel Photo Gallery with Categories**
Property photos organized into tabs: Rooms, Amenities, Dining, Exterior — reducing the undifferentiated scroll of a flat photo gallery.
Impact: Relevant photos found without scrolling 47 generic images — booking decision accelerated.

**#859 Review Summary with Scores**
Aggregated review scores broken out by dimension: Cleanliness, Location, Service, Value, Comfort — shown as bars alongside the overall score with a highlight of the highest-rated aspect.
Impact: What the property excels at instantly clear — decision-relevant signal extracted from review volume.

**#860 Multi-City / One-Way / Round-Trip Toggle**
A compact flight search form header with trip type toggle — switching between modes reconfigures the form fields (single return date vs. two departure dates vs. multi-segment form).
Impact: Search form matches intended trip structure — wrong-mode confusion eliminated.
