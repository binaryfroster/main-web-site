# 100 UI Elements Reference Library

Full reference for all 100 elements, organized by category.
Each entry includes: number, name, function, and UX impact.

---

## Category 1: Navigation & Wayfinding (#01–#10)

**#01 Sticky Smart Navbar**
Hides on scroll-down, reappears on scroll-up. Reclaims screen real estate on long pages while keeping nav always a flick away.
Impact: Keeps navigation accessible without obscuring content.

**#02 Mega Menu**
Expanded multi-column dropdown grouping related links with icons, descriptions, and featured items. Eliminates deep nested menus.
Impact: Users find content faster — reduced decision fatigue.

**#03 Breadcrumb Trail**
Hierarchical path indicator (Home → Products → Item). Improves orientation and offers one-click back navigation.
Impact: Eliminates "where am I?" confusion.

**#04 Reading Progress Bar**
Thin horizontal bar at viewport top filling as user scrolls through content. Sets expectations for content length.
Impact: Increases article completion rates.

**#05 Floating Table of Contents**
Sticky sidebar listing headings with active section highlighted. Allows instant jump to any section in long-form content.
Impact: Turns walls of text into scannable, navigable content.

**#06 Command Palette (⌘K)**
Keyboard-triggered modal for instant search-and-navigate. Power users jump to any page, action, or setting without mouse.
Impact: 10× navigation speed for power users in complex apps.

**#07 Active State Indicators**
Visual cues (highlighted item, underline, pill) on current page/section. Removes orientation confusion.
Impact: Users always know where they are.

**#08 Mobile Hamburger → Full-Screen Menu**
Hamburger icon expanding to full-viewport overlay with large tap targets and smooth animation.
Impact: Transforms clunky mobile nav into first-class experience.

**#09 Back-to-Top Button**
Floating button appearing after threshold scroll, enabling instant return to page top.
Impact: Saves users from exhausting scroll-back on long pages.

**#10 Step Indicator / Wizard Nav**
Numbered step bar for multi-step forms showing total steps, current step, and completion status.
Impact: Multi-step completion rates rise when the finish line is visible.

## Category 2: Forms & Input (#11–#20)

**#11 Floating Label Input**
Labels start as placeholder text and animate upward on focus. Saves vertical space while keeping context visible mid-typing.
Impact: Compact forms with zero label-amnesia.

**#12 Inline Validation**
Real-time feedback as users type — green checkmarks for valid, red underlines with specific messages for invalid.
Impact: Form errors drop dramatically; submission rates rise.

**#13 Password Strength Meter**
Color-coded progress bar showing password strength as user types. Guides toward stronger passwords without rigid blocking rules.
Impact: Users create stronger passwords — improved security and confidence.

**#14 Typeahead / Autocomplete**
Suggestions appear in a dropdown as user types, from past entries or dataset. Reduces keystrokes, prevents typos.
Impact: Search input effort cut by up to 60% for common queries.

**#15 OTP / PIN Input**
Individual character boxes for verification codes. Auto-advances focus, handles paste gracefully.
Impact: Verification completion rates soar with proper OTP UI.

**#16 Tag / Chip Input**
Multiple values (skills, tags, emails) entered as dismissable chips within a single field.
Impact: Multi-value entry feels intuitive instead of error-prone.

**#17 Date Picker / Calendar Widget**
Visual calendar popping from date input for click-to-select. Prevents format errors entirely.
Impact: Eliminates date-format errors.

**#18 Character Counter**
Live counter showing remaining characters, changing color as limit approaches.
Impact: Users self-edit more confidently — fewer truncated submissions.

**#19 Smart Address Lookup**
API-powered auto-fill of street, city, postcode from partial input (Google Places, Postcoder).
Impact: Checkout friction drops massively; cart abandonment falls.

**#20 Drag-and-Drop File Upload**
Dropzone accepting dragged files with previews, file names, sizes, and upload progress bars.
Impact: File upload becomes intuitive, visual, and confidence-building.

## Category 3: Buttons & CTAs (#21–#30)

**#21 Loading State Button**
Button transforms to loading spinner after click, preventing double-submits.
Impact: Eliminates duplicate submissions; builds process confidence.

**#22 Ripple Effect on Click**
Radial wave from click point confirming the button was registered.
Impact: Makes interfaces feel physically satisfying.

**#23 Split Button**
Primary action + dropdown arrow for alternative actions in one control.
Impact: Exposes options without cluttering UI with multiple buttons.

**#24 Success State Animation**
CTA morphs into checkmark or success animation after completion rather than blank redirect.
Impact: Delivers dopamine hit at the most critical conversion moment.

**#25 Sticky CTA Bar (Mobile)**
Fixed call-to-action bar anchored to bottom of mobile screen, persisting while scrolling.
Impact: Mobile conversions increase when CTA is always visible.

**#26 Icon + Label Button**
Buttons pairing recognizable icon with text label (🛒 Add to Cart, ↓ Download).
Impact: Scan-and-click speed increases; intent ambiguity eliminated.

**#27 Destructive Action Confirmation**
Secondary dialog or "hold to confirm" interaction for irreversible actions (delete, cancel).
Impact: Prevents accidental data loss — earns deep user trust.

**#28 Floating Action Button (FAB)**
Prominent circular button fixed in screen corner triggering primary action.
Impact: Primary action is always one tap away on any scroll position.

**#29 Toggle / Switch**
Sliding on/off toggle for binary settings with animated state change.
Impact: Settings feel immediate and physical — state understood instantly.

**#30 Ghost / Outline Button Pair**
Solid primary CTA paired with ghost secondary option creating visual hierarchy.
Impact: Guides users to preferred action without removing choice.

## Category 4: Feedback & Notifications (#31–#40)

**#31 Toast Notification**
Non-blocking auto-dismissing message sliding in from corner to confirm actions.
Impact: Confirms actions without navigating away — flow unbroken.

**#32 Undo Toast**
Toast with "Undo" button for 5–10 seconds after destructive actions.
Impact: Removes fear of mistakes — users act more confidently.

**#33 Skeleton Screen / Loading Placeholder**
Animated grey placeholder shapes matching layout of loading content.
Impact: Perceived load time drops — page feels "almost ready" immediately.

**#34 Empty State Illustration**
Friendly illustration and helpful message when lists or dashboards are empty.
Impact: First-time users don't bounce — they know what to do next.

**#35 Inline Error Messaging**
Specific, human-readable error messages directly below offending field.
Impact: Users fix errors faster — form completion rates rise.

**#36 Status Badge / Dot Indicator**
Small colored dots or pills conveying system status at a glance (green/red/yellow).
Impact: Status communicated in 1px without any text.

**#37 Notification Bell with Badge**
Bell icon with numbered badge showing unread count, tapping reveals activity dropdown.
Impact: Drives re-engagement and session length in dashboard products.

**#38 Alert / Banner Bar**
Full-width bar for critical announcements: maintenance windows, promotional deadlines.
Impact: Critical information reaches 100% of visitors.

**#39 Animated Progress Bar**
Real-time animated bar showing process completion (uploading, generating, processing).
Impact: Users wait 40% longer when they can see meaningful progress.

**#40 Confetti / Celebration Effect**
Animated confetti or particles triggered by key milestones (signup, first purchase, 100% profile).
Impact: Emotional peaks make products memorable and shareable.

## Category 5: Content Display (#41–#50)

**#41 Accordion / FAQ Section**
Collapsible sections that expand on click. Ideal for FAQs, feature lists, settings.
Impact: Information density without visual overwhelm.

**#42 Tabbed Interface**
Horizontal tabs switching between content panels without page reload.
Impact: Reduces page length while keeping all content accessible.

**#43 Tooltip / Popover**
Contextual hints on hover or focus over icons, abbreviations, or complex fields.
Impact: Delivers help exactly when needed without cluttering the base UI.

**#44 Image Lightbox / Gallery**
Click-to-open dark overlay at full size with prev/next navigation.
Impact: Product imagery examined fully — purchase confidence increases.

**#45 Code Syntax Highlighter**
Color-coded monospaced code blocks with language labels and one-click copy button.
Impact: Developer trust and comprehension skyrocket with proper code blocks.

**#46 Infinite Scroll with Sentinel**
Content loads automatically near bottom using Intersection Observer.
Impact: Session depth increases — users consume more content.

**#47 Comparison Table**
Structured grid comparing features across plans with checkmarks, crosses, highlighted recommended.
Impact: Decision-making time drops — users commit with more confidence.

**#48 Hover Card / Preview Card**
Hovering over a link shows contextual preview (name, avatar, bio) without navigating away.
Impact: Context without navigation — users stay in flow.

**#49 Virtualized List / Table**
Renders only visible rows of large datasets, recycling elements on scroll.
Impact: 10,000+ item lists scroll at 60fps — no browser lag.

**#50 Read More / Truncated Expandable Text**
Long text truncated after a line limit with "Read more" link.
Impact: Pages feel clean; detail available without endless scrolling.

## Category 6: Layout & Structure (#51–#60)

**#51 Responsive Grid System**
Fluid column grid adapting layout across screen sizes (4col → 2col → 1col).
Impact: One layout that works perfectly on every device.

**#52 Card Component**
Self-contained content units with shadow, border-radius, and padding.
Impact: Information hierarchy becomes instantly clear — scanning effortless.

**#53 Sidebar / Drawer Panel**
Panel sliding in from side showing secondary content without page change.
Impact: Secondary workflows accessible without losing primary context.

**#54 Modal Dialog**
Focused overlay window for specific actions (confirm, sign in, preview).
Impact: Task completion improves when users can't lose context mid-action.

**#55 Split-Screen Layout**
Two equal or proportional panels side by side for dual-context tasks.
Impact: Before/after, code+preview tasks become fluid.

**#56 Masonry / Waterfall Grid**
Pinterest-style variable-height columns packing content without gaps.
Impact: Visual content shows more per screen — immersive browsing.

**#57 Hero Section with CTA**
Full-viewport opening section with headline, supporting text, visual, and primary CTA.
Impact: First 3 seconds determine whether visitors stay or leave.

**#58 Sticky Header + Content Well**
Fixed top header with scrollable content and max-width container.
Impact: Reading comfortable on any screen width — optimal line length preserved.

**#59 Collapsible Sidebar Navigation**
Sidebar collapsing to icon-only mode to maximize content area.
Impact: Power users gain workspace; new users keep label context.

**#60 Resizable Panels (Drag to Resize)**
Draggable divider between panels for user-controlled layout.
Impact: Users personalize layout to their workflow — productivity increases.

## Category 7: Motion & Microinteractions (#61–#70)

**#61 Scroll-Triggered Fade In**
Content elements animate into view as they enter the viewport.
Impact: Scroll depth and engagement time increase with scroll animations.

**#62 Hover State Transitions**
Smooth CSS transitions (100–200ms) on color, scale, or shadow for interactive elements.
Impact: Highest perceived quality return for the smallest investment.

**#63 Page Transition Animation**
Smooth crossfade, slide, or morph between pages in a SPA.
Impact: Navigation feels continuous and spatial — more app-like.

**#64 Lottie Animation**
Lightweight JSON-based vector animations for onboarding, empty states, success moments.
Impact: Emotional storytelling at kilobyte file sizes.

**#65 Number Counter Animation**
Statistics animate from 0 to value when entering viewport.
Impact: Stats become dynamic proof points rather than forgettable text.

**#66 Parallax Scrolling Effect**
Background elements move at different scroll speed creating 3D depth illusion.
Impact: Visual depth makes a page feel premium and memorable.

**#67 Typewriter / Text Cycle Effect**
Headline cycling through multiple phrases with typing cursor animation.
Impact: Multiple messages delivered in one slot — eliminates headline compromise.

**#68 Cursor Follower / Trail Effect**
Custom cursor element following mouse with slight lag (spotlight glow, trailing dot).
Impact: Visitors spend more time mousing around — engagement increases.

**#69 Magnetic Button Effect**
Buttons subtly attracting toward cursor as it approaches via transform translations.
Impact: CTAs feel irresistible and memorable — unique physical interaction.

**#70 Staggered List Animation**
List items animate in sequentially with slight delay between each.
Impact: Transforms any list into a curated, intentional reveal.

## Category 9: Accessibility & Inclusivity (#81–#90)

**#81 Focus Ring / Visible Focus State**
Highly visible outline on keyboard-focused element.
Impact: Makes site usable for keyboard and assistive-tech users.

**#82 Skip to Main Content Link**
Visually hidden link visible on keyboard focus to skip repetitive navigation.
Impact: Essential for WCAG compliance; saves assistive-tech users from tab-fatigue.

**#83 Dark / Light Mode Toggle**
Theme switch persisting preference in localStorage.
Impact: 50%+ of users prefer dark mode — not offering it is a miss.

**#84 Font Size / Zoom Control**
Explicit A/A+ buttons for increasing/decreasing base font size.
Impact: Serves aging populations and visually impaired users.

**#85 ARIA Live Region**
Dynamically updated regions marked aria-live so screen readers announce changes.
Impact: Dynamic apps become fully usable with screen readers.

**#86 Color-Blind Friendly Palette**
Icons, patterns, or text labels alongside color — never color alone.
Impact: 8% of male users can now fully comprehend data visualizations.

**#87 Reduced Motion Mode**
Respecting prefers-reduced-motion CSS media query to minimize animations.
Impact: Inclusive animations — delight without triggering vestibular disorders.

**#88 High Contrast Mode**
Toggle or auto-detection switching to pure black/white high-contrast palette.
Impact: Low-vision users can finally read content clearly.

**#89 Accessible Modal Trap Focus**
Keyboard tab focus cycling within open modal, returning to trigger on close.
Impact: Modals usable for keyboard users — not just a visual trap.

**#90 Language Selector**
Dropdown or flag-based selector for UI language switching.
Impact: 72% of consumers prefer buying in their native language (CSA Research).

## Category 10: E-Commerce & Conversion (#91–#100)

**#91 Persistent Cart Icon with Count**
Cart icon in header with real-time item count update.
Impact: Keeps purchase intent visible — users return to complete the cart.

**#92 Product Image Zoom (Hover/Pinch)**
Magnified lens on hover, pinch zoom on mobile for product detail inspection.
Impact: Detailed inspection reduces return rates and increases conversions.

**#93 Stock Scarcity Indicator**
"Only 3 left!" or low-stock progress bar when inventory is genuinely limited.
Impact: Genuine scarcity messaging accelerates purchase decisions.

**#94 Wishlist / Save for Later**
Heart or bookmark icon on product cards saving items without adding to cart.
Impact: Captures intent from non-ready buyers — re-engaged later.

**#95 Price Breakdown Tooltip**
Info icon revealing detailed cost breakdown (subtotal, tax, shipping, discount) on hover.
Impact: No surprise costs — the leading cause of cart abandonment addressed.

**#96 Related / Recommended Products**
AI or rule-based "You might also like" or "Customers also bought" sections.
Impact: Amazon attributes 35% of revenue to its recommendation engine.

**#97 Countdown Timer**
Live ticking timer for limited-time offers, flash sales, or deal deadlines.
Impact: Conversion rates spike during countdown periods for genuine offers.

**#98 Free Shipping Progress Bar**
Cart-page indicator showing "Add $12 more for FREE shipping" with filling bar.
Impact: Users add items to cross threshold — proven AOV booster.

**#99 Guest Checkout Option**
Purchase without mandatory account creation; optional "save details" afterward.
Impact: Removing mandatory registration increased ASOS conversion by 50%.

**#100 Exit-Intent Popup**
Overlay triggered when cursor moves toward browser close, offering discount or resource.
Impact: Captures 10–15% of would-be exits with the right offer.
